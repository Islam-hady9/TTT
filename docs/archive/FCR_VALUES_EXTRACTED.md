## 📊 FCR (Feed Conversion Ratio) / معامل التحويل الغذائي

### Formula / المعادلة:

```
FCR = كمية العلف المستخدمة (بالطن) / كمية الأسماك المنتجة (بالطن)
```

**English**:
```
FCR = Total Feed Used (tons) / Total Fish Produced (tons)
```

---

## 📈 Interpretation / التفسير

**كل ما الرقم يقل → الكفاءة أعلى**

**The lower the number → The higher the efficiency**

---

## 🎯 What This Means / ماذا يعني هذا

### Good FCR Values / قيم FCR الجيدة:

- **Excellent / ممتاز**: FCR < 1.2
- **Good / جيد**: FCR = 1.2 - 1.5
- **Acceptable / مقبول**: FCR = 1.5 - 1.8
- **Poor / ضعيف**: FCR > 1.8

### Example / مثال:

If you use **100 kg of feed** and produce **80 kg of fish**:

```
FCR = 100 kg / 80 kg = 1.25
```

This means you need **1.25 kg of feed** to produce **1 kg of fish**.

## 🔗 Related Formula / معادلة ذات صلة

### 19. Feed Efficiency (FE) / كفاءة استخدام العلف

```
FE (%) = (الزيادة في الوزن / كمية العلف المستهلكة) × 100
```

**Note**: Feed Efficiency is the **inverse** of FCR  
**ملاحظة**: كفاءة استخدام العلف هي **عكس** معامل التحويل الغذائي

## 📋 All Formulas from Document / جميع المعادلات من الملف

### 1. Biomass / الكتلة الحيوية
```
Biomass = متوسط الوزن (كجم) × عدد الأسماك
```

### 2. Feeding Rate / معدل التغذية اليومي
```
معدل التغذية اليومي = Biomass × نسبة التغذية اليومية (%)
```

### 3. FCR / معامل التحويل الغذائي ⭐
```
FCR = كمية العلف المستخدمة (طن) / كمية الأسماك (طن)
```

### 4. Daily Growth Rate (DGR) / معدل النمو اليومي
```
DGR = (الوزن النهائي - الوزن الابتدائي) / عدد الأيام
```

### 5. Specific Growth Rate (SGR) / معدل النمو النسبي
```
SGR = ((الوزن النهائي - الوزن الابتدائي) / الزمن بالأيام) × 100
```

### 6. Stocking Density / كثافة التخزين
```
كثافة التخزين = عدد الأسماك أو Biomass (كجم) / حجم المياه (م³)
```

### 7. Oxygen Consumption / استهلاك الأكسجين
```
Oxygen Demand = Biomass × معدل استهلاك الأكسجين
غالباً: 200-500 mg O₂/kg/hour
```

### 8. Survival Rate / معدل البقاء
```
Survival Rate (%) = (عدد الأسماك الحية / عدد الأسماك المبدئي) × 100
```

### 9. Yield / إنتاجية الحوض
```
إنتاجية الحوض = إجمالي إنتاج الحوض (كجم) / مساحة أو حجم الحوض
```

### 10. Water Exchange Rate / معدل تغيير المياه
```
معدل تغيير المياه = حجم المياه المتغيرة يومياً / إجمالي حجم المياه
```

### 11. Condition Factor (K) / مؤشر حالة السمكة
```
K = (W / L³) × 100
W: الوزن (جم)
L: الطول (سم)
```

### 12. Mortality Rate / معدل النفوق
```
معدل النفوق (%) = (عدد الأسماك النافقة / العدد الكلي) × 100
```

### 13. Ammonia Production (TAN) / حمل الأمونيا
```
TAN = Feed × 0.03
(تقريباً 3% من العلف يتحول لأمونيا)
```

### 14. Flow Rate / معدل تدفق المياه
```
Q = V / T
Q: معدل التدفق
V: حجم الحوض
T: زمن التجديد
```

### 15. Biofilter Loading Rate / تحميل البيوفلتر
```
Loading = TAN / مساحة سطح البيوفلتر
```

### 16. Energy Efficiency / استهلاك الطاقة
```
Energy Efficiency = إجمالي الطاقة (kWh) / الإنتاج (kg)
```

### 17. Organic Loading / معامل التحميل العضوي
```
Organic Load = العلف اليومي / حجم النظام
```

### 18. Projected Harvest / الإنتاج المتوقع
```
Harvest = عدد الأسماك × متوسط الوزن المتوقع
```

### 19. Feed Efficiency (FE) / كفاءة استخدام العلف
```
FE (%) = (الزيادة في الوزن / كمية العلف المستهلكة) × 100
(عكس FCR)
```

### 20. Hydraulic Retention Time (HRT) / زمن بقاء المياه
```
HRT = V / Q
V: حجم المياه في الحوض
Q: كمية المياه المتغيرة في الساعة
```

### 21. Profit per kg / معدل الربحية لكل كجم
```
Profit/kg = سعر البيع - تكلفة الإنتاج
```

### 22. Productivity per m³ / معدل الإنتاج لكل وحدة ماء
```
Productivity = الإنتاج (kg) / حجم المياه (m³)
```

### 23. Actual Oxygen Consumption / معدل استهلاك الأكسجين الفعلي
```
O₂ = Biomass × Consumption Rate × عامل النشاط
(عامل النشاط يتغير حسب: درجة الحرارة، التغذية، حركة السمك)
```

### 24. Turnover Rate / معدل تجديد الحوض
```
Turnover Rate = Q / V
Q: معدل تدفق المياه
V: حجم الحوض
```
