## 🔄 دورة حياة الأسماك الكاملة / Complete Fish Lifecycle

### المراحل الأربعة / Four Stages

```
1. وحدة التحضين (Nursery Unit)
   ├── البيض (Eggs): 0 - 0.001g
   ├── اليرقات (Fry): 0.001 - 0.1g
   └── الصغار (Fingerlings): 0.1 - 1g

2. وحدة التربية (Pregrow Unit)
   ├── النمو (Juveniles): 1 - 30g
   └── الأسماك الصغيرة (Young Fish): 30 - 200g

3. وحدة التسمين (Grow out Unit)
   └── التسمين (Fattening): 200g - 600g

4. الحصاد (Harvest) ⭐ NEW
   └── جاهز للسوق (Market Ready): 350 - 600g
       ├── جاهز (Ready): 350 - 400g
       └── مثالي (Optimal): 400 - 600g
```

- 📈 **متوسط FCR** (Avg FCR): متوسط معامل التحويل الغذائي

### 2. تبويبات منظمة / Organized Tabs

**تبويبان رئيسيان:**
- ✅ **جاهز للحصاد** (Ready for Harvest): الدفعات التي وصلت لوزن السوق
- ✔️ **تم الحصاد** (Completed): سجل عمليات الحصاد المكتملة

### 3. نظام تصفية وترتيب / Filtering & Sorting System

**خيارات الترتيب:**
- 📏 **حسب الوزن** (By Weight): من الأثقل للأخف
- 📅 **حسب العمر** (By Age): من الأقدم للأحدث
- 📊 **حسب FCR** (By FCR): من الأفضل للأسوأ

### 4. مؤشرات حالة الحصاد / Harvest Status Indicators

**3 حالات:**
- 🟢 **مثالي** (Optimal): 400 - 600g - أفضل وزن للسوق
- 🔵 **جاهز** (Ready): 350 - 400g - جاهز للحصاد
- ⚪ **غير جاهز** (Not Ready): < 350g - يحتاج مزيد من الوقت

### 5. بطاقات تفصيلية للدفعات / Detailed Batch Cards

**كل بطاقة تعرض:**
- 🏷️ كود الدفعة (Batch Code)
- 🏊 رقم الحوض (Pond ID)
- 🐟 عدد الأسماك (Fish Count)
- ⚖️ متوسط الوزن (Average Weight)
- 📦 الكتلة الحية (Biomass)
- 📅 عمر الدفعة (Age in Days)
- 📊 معامل التحويل الغذائي مع مؤشر ملون (FCR with Color Indicator)
- 🎯 حالة الحصاد (Harvest Status)

### 6. إجراءات سريعة / Quick Actions

**أزرار الإجراءات:**
