## 📊 المراحل الأربعة الجديدة

### 1️⃣ **الزريعة (Fry)** 🥚
**الوزن:** 0 - 0.5 جرام  
**المدة:** 0-14 يوم  
**المكان:** **وحدة التحضين (Nursery Unit)**  
**الوصف:** تجميع البيض + اليرقات

```python
0g ≤ الوزن < 0.5g → fry (زريعة)
```

**التغذية:**
- معدل التغذية: 15-18% من الكتلة الحية
- عدد الوجبات: 6 وجبات يومياً
- نوع العلف: Starter (بادئ)

---

### 2️⃣ **الإصبعيات (Fingerlings)** 🐠
**الوزن:** 0.5 - 40 جرام  
**المدة:** 14-45 يوم  
**المكان:** **وحدة التحضين (Nursery Unit)**  
**الوصف:** متوسط 0.5 جرام في التحضين

```python
0.5g ≤ الوزن < 40g → fingerlings (إصبعيات)
```

**التغذية:**
- معدل التغذية: 10-15% من الكتلة الحية
- عدد الوجبات: 4 وجبات يومياً
- نوع العلف: Starter (بادئ)

**⚠️ نقطة النقل:** عند وصول الوزن إلى **40 جرام** → النقل إلى **وحدة التربية**

---

### 3️⃣ **التربية (Juveniles/Pregrow)** 🐟
**الوزن:** 40 - 200 جرام  
**المدة:** 45-90 يوم  
**المكان:** **وحدة التربية (Pregrow Unit)**  
**الوصف:** متوسط 40 جرام بداية مرحلة التربية

```python
40g ≤ الوزن < 200g → juveniles (تربية)
```

**التغذية:**
- معدل التغذية: 5-10% من الكتلة الحية
- عدد الوجبات: 3 وجبات يومياً
- نوع العلف: Grower (نمو)

**⚠️ نقطة النقل:** عند وصول الوزن إلى **200 جرام** → النقل إلى **وحدة التسمين**

---

### 4️⃣ **التسمين (Fattening/Grow out)** 🐠
**الوزن:** 200+ جرام  
**المدة:** 90-150 يوم  
**المكان:** **وحدة التسمين (Grow out Unit)**  
**الوصف:** المرحلة النهائية قبل الحصاد

```python
الوزن ≥ 200g → fattening (تسمين)
```

**التغذية:**
- معدل التغذية: 1-3% من الكتلة الحية
- عدد الوجبات: 2 وجبات يومياً
- نوع العلف: Fattening (تسمين)

**🎯 الحصاد:** عند 350-600 جرام (الوزن المثالي: 400-600 جرام)
