## 🔬 الحسابات العلمية

### 1. الكتلة الحية (Biomass)
```
Biomass (kg) = current_count × avg_weight (g) / 1000
```

### 2. معامل التحويل الغذائي (FCR)
```
FCR = total_feed_consumed (kg) / weight_gained (kg)

Classifications:
- Excellent: FCR < 1.5
- Good: 1.5 ≤ FCR ≤ 1.8
- Poor: FCR > 1.8
```

### 3. معدل النمو النوعي (SGR)
```
SGR (%) = ((ln(W2) - ln(W1)) / days) × 100

Classifications:
- Excellent: SGR > 7%
- Good: 5% ≤ SGR ≤ 7%
- Poor: SGR < 5%
```

### 4. معدل النفوق (Mortality Rate)
```
Mortality Rate (%) = (total_mortality / initial_count) × 100

Thresholds:
- Hatchery: 35%
- Growout: 10%
- Fattening: 10%
```

### 5. التنبؤ بالوزن (Weight Prediction)
```
Predicted Weight (g) = current_weight × e^(SGR × days / 100)
```

### 6. التنبؤ بتاريخ الحصاد (Harvest Date Prediction)
```
Days Remaining = (ln(target_weight) - ln(current_weight)) / (SGR / 100)
```

### 7. كمية العلف اليومية (Daily Feed Amount)
```
Daily Feed (kg) = biomass (kg) × feeding_rate

Feeding Rates by Stage:
- Fry: 15-18% of biomass
- Fingerlings: 10-15% of biomass
- Juveniles: 5-10% of biomass
- Young Fish: 3-5% of biomass
- Fattening: 1-3% of biomass
```

## 🚨 التنبيهات التلقائية

### عتبات التنبيهات:
1. **FCR Warning**: FCR > 1.8
2. **SGR Warning**: SGR < 5.0%
3. **Mortality Hatchery**: > 35%
4. **Mortality Growout/Fattening**: > 10%
5. **Transfer Ready (Fingerlings)**: avg_weight ≥ 1.0g
6. **Transfer Ready (Young Fish)**: avg_weight ≥ 200g
7. **Harvest Ready**: 350g ≤ avg_weight ≤ 600g
8. **Water Quality**: خارج النطاق الأمثل
