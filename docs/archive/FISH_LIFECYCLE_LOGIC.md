# 🐟 منطق دورة حياة السمكة الكاملة (Fish Lifecycle Logic)

## 📋 نظرة عامة

هذا المستند يشرح المنطق الكامل لتتبع دورة حياة السمكة من البيضة إلى الحصاد، مع جميع الحسابات والعلاقات.

---

## 🔄 مراحل دورة الحياة (Life Cycle Stages)

### 1. **البيض (Eggs)** - 0-3 أيام
- **الوزن**: < 0.001 جم
- **العلف**: لا يوجد
- **الحوض**: Hatchery
- **المدة**: 2-3 أيام

### 2. **اليرقات (Fry)** - 3-14 يوم
- **الوزن**: 0.001 - 0.1 جم
- **العلف**: Starter (< 1g)
- **معدل التغذية**: 15-18% من الكتلة الحية
- **عدد مرات التغذية**: 4-6 مرات يومياً
- **الحوض**: Hatchery
- **المدة**: 10-14 يوم

### 3. **الإصبعيات الصغيرة (Fingerlings)** - 14-30 يوم
- **الوزن**: 0.1 - 1 جم
- **العلف**: Starter (< 1g)
- **معدل التغذية**: 10-15% من الكتلة الحية
- **عدد مرات التغذية**: 4 مرات يومياً
- **الحوض**: Hatchery → Growout (عند 1 جم)
- **المدة**: 15-20 يوم

### 4. **الإصبعيات (Juveniles)** - 30-60 يوم
- **الوزن**: 1 - 30 جم
- **العلف**: Grower (1-30g)
- **معدل التغذية**: 5-10% من الكتلة الحية
- **عدد مرات التغذية**: 3-4 مرات يومياً
- **الحوض**: Growout
- **المدة**: 30-40 يوم

### 5. **الأسماك الصغيرة (Young Fish)** - 60-90 يوم
- **الوزن**: 30 - 200 جم
- **العلف**: Finisher (30-200g)
- **معدل التغذية**: 3-5% من الكتلة الحية
- **عدد مرات التغذية**: 2-3 مرات يومياً
- **الحوض**: Growout → Fattening (عند 200 جم)
- **المدة**: 30-40 يوم

### 6. **التسمين (Fattening)** - 90-150 يوم
- **الوزن**: 200 - 500+ جم
- **العلف**: Fattening (> 200g)
- **معدل التغذية**: 1-3% من الكتلة الحية
- **عدد مرات التغذية**: 2 مرات يومياً
- **الحوض**: Fattening
- **المدة**: 60-90 يوم
- **الحصاد**: عند 400-500 جم

## 📊 الحسابات الأساسية

### 1. **الكتلة الحية (Biomass)**
```
Biomass = عدد الأسماك الحالي × متوسط الوزن
```

**مثال**:
- عدد الأسماك: 10,000
- متوسط الوزن: 50 جم
- Biomass = 10,000 × 50 = 500,000 جم = 500 كجم

**التحديث**:
- يتم تحديث Biomass بعد كل عملية نفوق
- يتم تحديث Biomass بعد كل عملية وزن

---

### 2. **كمية العلف اليومية (Daily Feed Amount)**
```
كمية العلف = Biomass × معدل التغذية
```

**مثال**:
- Biomass: 500 كجم
- معدل التغذية: 3% (للأسماك 30-200 جم)
- كمية العلف = 500 × 0.03 = 15 كجم/يوم

**توزيع على الوجبات**:
- إذا كانت 3 وجبات: 15 ÷ 3 = 5 كجم/وجبة
- إذا كانت 4 وجبات: 15 ÷ 4 = 3.75 كجم/وجبة

---

### 3. **FCR (Feed Conversion Ratio)**
```
FCR = إجمالي العلف المستهلك / إجمالي الوزن المكتسب
```

**مثال**:
- إجمالي العلف: 1000 كجم
- الوزن الابتدائي: 100 كجم
- الوزن النهائي: 700 كجم
- الوزن المكتسب: 700 - 100 = 600 كجم
- FCR = 1000 / 600 = 1.67

**التفسير**:
- FCR = 1.67 يعني: كل 1.67 كجم علف ينتج 1 كجم سمك
- FCR المثالي للبلطي: 1.2 - 1.8
- FCR < 1.5 ممتاز
- FCR 1.5 - 1.8 جيد
- FCR > 1.8 يحتاج تحسين

---

### 4. **SGR (Specific Growth Rate)**
```
SGR = ((ln(الوزن النهائي) - ln(الوزن الابتدائي)) / عدد الأيام) × 100
```

**مثال**:
- الوزن الابتدائي: 1 جم
- الوزن النهائي: 50 جم
- عدد الأيام: 30 يوم
- SGR = ((ln(50) - ln(1)) / 30) × 100
- SGR = ((3.912 - 0) / 30) × 100
- SGR = 13.04% يومياً

**التفسير**:
- SGR > 10% ممتاز
- SGR 5-10% جيد
- SGR < 5% ضعيف

---

### 5. **معدل النفوق (Mortality Rate)**
```
معدل النفوق = (إجمالي النفوق / العدد الابتدائي) × 100
```

**مثال**:
- العدد الابتدائي: 10,000
- إجمالي النفوق: 500
- معدل النفوق = (500 / 10,000) × 100 = 5%

**التفسير**:
- معدل النفوق < 5% ممتاز
- معدل النفوق 5-10% مقبول
- معدل النفوق > 10% مشكلة (يحتاج تدخل)

---

### 6. **معدل البقاء (Survival Rate)**
```
معدل البقاء = ((العدد الحالي / العدد الابتدائي) × 100)
```

**مثال**:
- العدد الابتدائي: 10,000
- العدد الحالي: 9,500
- معدل البقاء = (9,500 / 10,000) × 100 = 95%

---

### 7. **توقع الوزن (Weight Prediction)**
```
الوزن المتوقع = الوزن الحالي × e^(SGR × عدد الأيام / 100)
```

**مثال**:
- الوزن الحالي: 50 جم
- SGR: 10% يومياً
- عدد الأيام: 30 يوم
- الوزن المتوقع = 50 × e^(10 × 30 / 100)
- الوزن المتوقع = 50 × e^3
- الوزن المتوقع = 50 × 20.09 = 1004.5 جم

---

### 8. **توقع تاريخ الحصاد (Harvest Date Prediction)**
```
الأيام المتبقية = (ln(الوزن المستهدف) - ln(الوزن الحالي)) / (SGR / 100)
```

**مثال**:
- الوزن الحالي: 50 جم
- الوزن المستهدف: 400 جم
- SGR: 10% يومياً
- الأيام المتبقية = (ln(400) - ln(50)) / 0.10
- الأيام المتبقية = (5.99 - 3.91) / 0.10
- الأيام المتبقية = 20.8 يوم ≈ 21 يوم

## 🔄 سير العمل الكامل (Complete Workflow)

### **المرحلة 1: إنشاء دورة جديدة (New Batch)**

```
1. إنشاء Batch جديد
   ├─ batch_code: "B001"
   ├─ pond_id: 1 (Hatchery)
   ├─ stocking_date: "2026-05-08"
   ├─ initial_count: 10,000
   ├─ current_count: 10,000
   ├─ avg_weight: 0.001 جم (بيض)
   ├─ stage: "eggs"
   └─ status: "active"

2. حساب Biomass الابتدائي
   Biomass = 10,000 × 0.001 = 10 جم = 0.01 كجم
```

---

### **المرحلة 2: التغذية اليومية**

```
1. حساب كمية العلف
   ├─ Biomass الحالي: 500 كجم
   ├─ معدل التغذية: 3% (حسب المرحلة)
   └─ كمية العلف = 500 × 0.03 = 15 كجم/يوم

2. تسجيل التغذية
   ├─ pond_id: 1
   ├─ feed_amount: 5 كجم (وجبة واحدة)
   ├─ feed_type: "grower"
   ├─ feeding_time: "08:00"
   ├─ fed_by: "أحمد"
   └─ fed_at: "2026-05-08 08:00:00"

3. تحديث إجمالي العلف المستهلك
   total_feed += 5 كجم
```

---

### **المرحلة 3: تسجيل النفوق**

```
1. تسجيل النفوق
   ├─ pond_id: 1
   ├─ mortality_count: 50
   ├─ cause: "water_quality"
   └─ recorded_at: "2026-05-08"

2. تحديث العدد الحالي
   current_count = 10,000 - 50 = 9,950

3. حساب معدل النفوق
   mortality_rate = (50 / 10,000) × 100 = 0.5%

4. تحديث Biomass
   Biomass = 9,950 × 50 = 497,500 جم = 497.5 كجم
```

---

### **المرحلة 4: قياس جودة المياه**

```
1. تسجيل القراءات
   ├─ pond_id: 1
   ├─ do: 7.2 mg/L
   ├─ ph: 7.8
   ├─ temperature: 28°C
   ├─ tan: 0.3 mg/L
   ├─ alkalinity: 135 mg/L
   └─ floc: 32

2. التحقق من النطاقات
   ├─ DO: 7.2 ✅ (6.0-8.0 مثالي)
   ├─ pH: 7.8 ✅ (7.0-8.3 مثالي)
   ├─ Temperature: 28°C ✅ (25-30 مثالي)
   └─ TAN: 0.3 ✅ (0-0.5 مثالي)

3. إنشاء تنبيه إذا خارج النطاق
   إذا DO < 6.0 أو > 8.0:
   └─ تنبيه: "جودة المياه خارج النطاق المثالي"
```

---

### **المرحلة 5: الوزن الدوري (Sampling)**

```
1. أخذ عينة
   ├─ عدد الأسماك في العينة: 50
   ├─ إجمالي وزن العينة: 2,500 جم
   └─ متوسط الوزن = 2,500 / 50 = 50 جم

2. تحديث متوسط الوزن في Batch
   avg_weight = 50 جم

3. تحديث Biomass
   Biomass = 9,950 × 50 = 497,500 جم = 497.5 كجم

4. حساب SGR
   SGR = ((ln(50) - ln(1)) / 30) × 100 = 13.04%

5. توقع الوزن بعد 30 يوم
   الوزن المتوقع = 50 × e^(13.04 × 30 / 100) = 2,245 جم
```

---

### **المرحلة 6: النقل بين الأحواض (Transfer)**

```
1. شروط النقل
   ├─ من Hatchery إلى Growout: عند 1 جم
   ├─ من Growout إلى Fattening: عند 200 جم
   └─ من Fattening إلى الحصاد: عند 400-500 جم

2. عملية النقل
   ├─ batch_id: 1
   ├─ from_pond_id: 1 (Hatchery)
   ├─ to_pond_id: 2 (Growout)
   ├─ transfer_count: 9,950
   ├─ avg_weight: 1 جم
   └─ transfer_date: "2026-06-07"

3. تحديث Batch
   ├─ pond_id: 2 (الحوض الجديد)
   ├─ stage: "juveniles"
   └─ status: "active"

4. تسجيل في History
   ├─ action: "transfer"
   ├─ from_pond: "H001"
   ├─ to_pond: "G001"
   └─ date: "2026-06-07"
```

---

### **المرحلة 7: الحصاد (Harvest)**

```
1. شروط الحصاد
   ├─ الوزن: 400-500 جم
   ├─ العمر: 120-150 يوم
   └─ FCR: < 1.8

2. عملية الحصاد
   ├─ batch_id: 1
   ├─ harvest_count: 9,500
   ├─ avg_weight: 450 جم
   ├─ total_weight: 9,500 × 450 = 4,275,000 جم = 4,275 كجم
   └─ harvest_date: "2026-10-05"

3. حساب المؤشرات النهائية
   ├─ FCR = total_feed / (total_weight - initial_weight)
   ├─ معدل البقاء = (9,500 / 10,000) × 100 = 95%
   ├─ SGR = ((ln(450) - ln(0.001)) / 150) × 100 = 8.7%
   └─ مدة الدورة: 150 يوم

4. تحديث Batch
   ├─ status: "harvested"
   ├─ harvest_date: "2026-10-05"
   └─ final_count: 9,500

5. إنشاء تقرير الدورة الكاملة
```

## 📈 تتبع التاريخ (History Tracking)

### **جدول Batch History**
```sql
CREATE TABLE batch_history (
    id INTEGER PRIMARY KEY,
    batch_id INTEGER,
    action VARCHAR,  -- 'created', 'transfer', 'sampling', 'harvest'
    from_pond_id INTEGER,
    to_pond_id INTEGER,
    count INTEGER,
    avg_weight FLOAT,
    notes TEXT,
    created_at TIMESTAMP
);
```

### **الأحداث المسجلة**:
1. **إنشاء الدورة** (created)
2. **النقل** (transfer)
3. **الوزن** (sampling)
4. **الحصاد** (harvest)

## 🎯 المؤشرات الرئيسية (KPIs)

### **على مستوى الدورة (Batch Level)**
1. FCR (Feed Conversion Ratio)
2. SGR (Specific Growth Rate)
3. معدل البقاء (Survival Rate)
4. مدة الدورة (Cycle Duration)
5. الوزن النهائي (Final Weight)
6. الإنتاج الكلي (Total Production)

### **على مستوى الحوض (Pond Level)**
1. Biomass الحالي
2. متوسط جودة المياه
3. معدل التغذية اليومي
4. معدل النفوق الشهري

### **على مستوى المزرعة (Farm Level)**
1. إجمالي الإنتاج الشهري
2. متوسط FCR للمزرعة
3. معدل استخدام الأحواض
4. الربحية
